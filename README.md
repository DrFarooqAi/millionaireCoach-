## What It Does

- Interviews you about your background and skills
- Finds your best niche using the proven framework
- Builds your **First-Draft Offer** using the 6 Ps (Person → Problem → Promise → Plan → Product → Price)
- Gives you clickable options at key decision points — no blank-page anxiety
- Downloads your personalized **Offer One-Pager** when done
- Tells you exactly what to do next (Phase 2: Validation)

**Session takes ~20–30 minutes. Completely free.**

## Setup (run it yourself)

### Step 1 — Get a free Groq API key
Go to [console.groq.com](https://console.groq.com) → sign up → create an API key (takes 2 minutes, no credit card).

### Step 2 — Install & run
```bash
pip install streamlit groq
python -m streamlit run app.py
```

Browser opens automatically. Paste your Groq key in the sidebar → start coaching.

## Deploy to Streamlit Cloud (share a link)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io) → connect your repo → deploy
3. Share the `.streamlit.app` URL with your audience

No secrets needed in deployment — every user brings their own free Groq key.
